--- layout: post title: LEAF Bering-uClibc 4.0 beta2 released author: kapeka --- As of today the second beta of Bering-uClibc 4.0 is availabe for download. It contains kernel and package updates and bugfixes for beta1. Most notable changes are:
  • The Linux kernel has been updated to 2.6.35.10
  • busybox has been updated to 1.18.2
  • Introducing shorewall-lite and shorewall6-lite
  • A new image Bering-uClibc_i486_syslinux_ser (e.g for WRAP boards) is provided
  • Bootstrapping into tmpfs for rootfs again (like Bering-uClibc3 releases); default size for root is 40MB
  • Added missing lwp files for webconf
See also

Bering-uClibc_4.x_-_Changelog

for a complete Changelog.

The available images are:

  • Bering-uClibc_4.0-beta2_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.0-beta2_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.0-beta2_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.0-beta2_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.0-beta2_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.0-beta2_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards
  • Bering-uClibc_4.0-beta2_src.tar - the complete sources tarball (as required by SF)
You'll find all these in FRS: https://sourceforge.net/projects/leaf/files/
