--- layout: post title: Release of the second beta version of Bering-uClibc 4.2 author: kapeka --- Bering-uClibc 4.2-beta2 is available for testing in the SF Files Area. It provides fixes for bugs and annoyances reported by users of the first beta version. Improves support for VIA Eden boards and updates a few packages to latest upstream versions. See Bering-uClibc_4.x_-_Changelog for a complete Changelog. The available images are:
  • Bering-uClibc_4.2-beta2_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.2-beta2_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.2-beta2_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.2-beta2_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.2-beta2_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.2-beta2_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards)
  • Bering-uClibc_4.2-beta2_src.tar - the complete sources tarball (as required by SF)
You'll find all these at: https://sourceforge.net/projects/leaf/files/
