--- layout: post title: The first beta of Bering-uClibc 4.1 is available for testing author: kapeka --- The first beta version of Bering-uClibc 4.1 introduces a kernel update, updates of various packages to new upstream versions, general improvements to support more harwdare right from the start and, as usual, some few fixes for bugs found in the latest version. Also the following new packages has been added to the distribution:
  • bind (http://www.isc.org/software/bind)
  • openldap client library
  • dibbler-server (http://klub.com.pl/dhcpv6/)
  • mbusd (http://mbus.sourceforge.net/)
  • sysstat
And last but not least webconf has been updated to work with newer versions of haserl and support for tooltips. This will enhance the capabilities to provide a web-based configuration tool. See Bering-uClibc_4.x_-_Changelog for a complete Changelog. The available images are:
  • Bering-uClibc_4.1-beta1_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.1-beta1_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.1-beta1_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.1-beta1_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.1-beta1_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.1-beta1_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards)
  • Bering-uClibc_4.1-beta1_src.tar - the complete sources tarball (as required by SF)
You'll find all these at: https://sourceforge.net/projects/leaf/files/
