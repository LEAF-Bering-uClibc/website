--- layout: post title: LEAF Bering-uClibc 4.0.1 available author: kapeka --- LEAF Bering-uClibc 4.0.1 is a refreshment of Bering-uClibc 4.0 images and to support more systems out of the box. Though it also fixes a few reported problems, there is roughly no need to update, if you have already installed 4.0. See also

Bering-uClibc_4.x_-_Changelog

for a complete Changelog.

The available images are:

  • Bering-uClibc_4.0.1_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.0.1_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.0.1_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.0.1_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.0.1_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.0.1_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards
  • Bering-uClibc_4.0.1_src.tar - the complete sources tarball (as required by SF)
You'll find all these in FRS: https://sourceforge.net/projects/leaf/files/
