--- layout: post title: LEAF Bering-uClibc 4.1.1 released author: kapeka --- As of today the Bering-uClicb team releases version 4.1.1. The only change from RC1 is the addition of the busybox watch applet.

See Bering-uClibc_4.x_-_Changelog for a complete Changelog. The available images are:

  • Bering-uClibc_4.1.1_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.1.1_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.1.1_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.1.1_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.1.1_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.1.1_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards)
  • Bering-uClibc_4.1.1_src.tar - the complete sources tarball (as required by SF)
You'll find all these at: https://sourceforge.net/projects/leaf/files/
