--- layout: post title: First release candidate of LEAF Bering-uClibc 4.3 available author: kapeka --- LEAF Bering-uClibc-4.3-rc1 provides package updates for quagga, radvd, pmacctd, tor and bird. Also bird has been fixed to run on ipv4 as well as on ipv6. Support to boot from USB devices has been fixed and improved. The kernel has now builtin crc32, enabled support to detect memory leak detection and for the 686 kernel the memory management has been improved. Closed Trac Tickets in this release: #77, #78 and #71 See Bering-uClibc_4.x_-_Changelog for a complete Changelog. The available images are:
  • Bering-uClibc_4.3-rc1_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.3-rc1_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.3-rc1_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.3-rc1_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.3-rc1_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.3-rc1_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards)
  • Bering-uClibc_4.3-rc1_src.tar - the complete sources tarball (as required by SF)
You'll find all these at: https://sourceforge.net/projects/leaf/files/
