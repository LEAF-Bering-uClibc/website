--- layout: post title: LEAF Bering-uClibc 4.2 released author: kapeka --- LEAF Beringg-uClibc 4.2 is a maintenance release for Bering-uClibc 4.x series. It's main feature is a kernel configuration review; the i686 kernel has been optimized for speed, while the geode and i486 versions require less space. Also the support for IPv6 and IPSEC has been improved.

LEAF Bering-uClibc 4.2 also provides package updates (e.g. shorewall, openvpn, openssl, openswan,...) and introduces a some new packages, like xtables-addons and iperf. Also included is support for Zeroconf using avahi.

See Bering-uClibc_4.x_-_Changelog for a complete Changelog. The available images are:

  • Bering-uClibc_4.2_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.2_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.2_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.2_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.2_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.2_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards)
  • Bering-uClibc_4.2_src.tar - the complete sources tarball (as required by SF)
You'll find all these at: https://sourceforge.net/projects/leaf/files/
