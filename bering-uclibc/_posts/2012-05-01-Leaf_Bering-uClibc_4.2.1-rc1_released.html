--- layout: post title: Leaf Bering-uClibc 4.2.1-rc1 released author: kapeka --- LEAF Bering-uClibc 4.2.1-rc1 provides new upstream versions for lmsensors, radvd, vsftpd, curl and openssl.libiconv.lrp has been added to the the list availbale packages. Additionally an annoying feedback message while saving configs should be fixed now.

See Bering-uClibc_4.x_-_Changelog for a complete Changelog. The available images are:

  • Bering-uClibc_4.2.1-rc1_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.2.1-rc1_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.2.1-rc1_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.2.1-rc1_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.2.1-rc1_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.2.1-rc1_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards)
  • Bering-uClibc_4.2.1-rc1_src.tar - the complete sources tarball (as required by SF)
You'll find all these at: https://sourceforge.net/projects/leaf/files/
