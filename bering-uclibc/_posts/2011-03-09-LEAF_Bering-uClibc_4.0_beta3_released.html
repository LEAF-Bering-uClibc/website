--- layout: post title: LEAF Bering-uClibc 4.0 beta3 released author: kapeka --- As of today the third beta of Bering-uClibc 4.0 is availabe for download. It contains kernel and package updates and bugfixes for beta2. Most notable changes are:
  • The Linux kernel has been updated to 2.6.35.11
  • busybox applet ifconfig has been added, fixes openswan errors
  • finally fixing date format errors in shorewall log files
  • fixed apkg merge (Trac ticket #29)
  • fixed tcpdump for ipv6
  • fixed daemontools (Trac ticket #18)
See also

Bering-uClibc_4.x_-_Changelog

for a complete Changelog.

The available images are:

  • Bering-uClibc_4.0-beta3_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.0-beta3_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.0-beta3_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.0-beta3_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.0-beta3_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.0-beta3_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards
  • Bering-uClibc_4.0-beta3_src.tar - the complete sources tarball (as required by SF)
You'll find all these in FRS: https://sourceforge.net/projects/leaf/files/
