--- layout: post title: LEAF Bering-uClibc 4.1-beta2 released author: kapeka --- The second beta version of LEAF Bering-uClibc 4.1 provides an update of the kernel to 2.6.35.14, new upstream versions of busybox (1.19), tinyproxy (1.8.3) and dnsmasq (2.58) and fixes for webconf, systat,mbusd and ppp-hsh.

The nut package received fixes for environment and scripts and a patch for Powercom devices. The user/group "nut" has been added to /etc/passwd and /etc/group in etc.lrp to let run "nut" with user "nut" instead of user "root".

See Bering-uClibc_4.x_-_Changelog for a complete Changelog. The available images are:
  • Bering-uClibc_4.1-beta2_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.1-beta2_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.1-beta2_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.1-beta2_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.1-beta2_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.1-beta2_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards)
  • Bering-uClibc_4.1-beta2_src.tar - the complete sources tarball (as required by SF)
You'll find all these at: https://sourceforge.net/projects/leaf/files/
