--- layout: post title: LEAF Bering-uClibc 4.0 Release Candidate 1 released author: kapeka --- The first (and probably only) release candidate of Bering-uClibc 4.0 is availabe for download. As expected in the late release stage Bering-uClibc 4.0-rc1 provides mostly minor package updates, a few bugfixes and finetuning.

Most notable changes are:

  • enhanced hardware detection in hwdetect (Trac ticket #13)
  • added ip6tables to all images by default (Trac ticket #41)
  • get yate to work (Trac ticket #35)
  • more openswan fixes

See also

Bering-uClibc_4.x_-_Changelog

for a complete Changelog.

The available images are:

  • Bering-uClibc_4.0-rc1_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.0-rc1_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.0-rc1_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.0-rc1_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.0-rc1_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.0-rc1_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards
  • Bering-uClibc_4.0-rc1_src.tar - the complete sources tarball (as required by SF)
You'll find all these in FRS: https://sourceforge.net/projects/leaf/files/
