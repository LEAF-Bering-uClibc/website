--- layout: post title: First release candidate of Bering-uCilbc 4.2 released author: kapeka --- The first release candidate for Bering-uClibc 4.2 is available for testing. It provides a security update of dropbear, an update for ethtool and new packages for ahavi support and iperf.

Please note that an issue with avahi's pid file has been fixed after Bering-uClibc 4.2-rc1 has been tagged. An updated avahi package is available via the Packages 4.x page.

See Bering-uClibc_4.x_-_Changelog for a complete Changelog. The available images are:

  • Bering-uClibc_4.2-rc1_geode_syslinux_ser.tar.gz - optimized for Geode CPU and seriell interface (e.g. ALIX Boards)
  • Bering-uClibc_4.2-rc1_i686_isolinux_vga.iso - CD image optimized for i686 CPU with VGA interface
  • Bering-uClibc_4.2-rc1_i486_isolinux_vga.iso - CD image optimized for i486 CPU with VGA interface
  • Bering-uClibc_4.2-rc1_i686_syslinux_vga.tar.gz - optimized for i686 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.2-rc1_i486_syslinux_vga.tar.gz - optimized for i486 CPU with VGA interface to load from hd/cf
  • Bering-uClibc_4.2-rc1_i486_syslinux_ser.tar.gz - optimized for i486 CPU with serial interface (e.g. WRAP boards)
  • Bering-uClibc_4.2-rc1_src.tar - the complete sources tarball (as required by SF)
You'll find all these at: https://sourceforge.net/projects/leaf/files/
